Numbers

Basic Arithmetic – Study Material

What are Numbers?

Numbers are mathematical values used for counting, measuring, and performing calculations.

Types of Numbers

  • Natural Numbers: 1, 2, 3, 4, ...
  • Whole Numbers: 0, 1, 2, 3, ...
  • Integers: ..., -2, -1, 0, 1, 2, ...

Number System

The number system represents numbers using digits. Example: Decimal system (Base 10).

Place Value & Face Value

Place Value: Value based on position (e.g., 5 in 50 = 50).
Face Value: Actual digit value (e.g., 5 in 50 = 5).

Basic Operations

  • Addition (+)
  • Subtraction (−)
  • Multiplication (×)
  • Division (÷)

BODMAS / PEMDAS Rule

Order of operations: Brackets → Orders → Division → Multiplication → Addition → Subtraction
